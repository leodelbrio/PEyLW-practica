INTRODUCCION HTML

Este es un curso que prendente introduccir al lector en el mundo del HTML. No se requieren conocimientos previos, se empezará desde cero. No hay que saber nada de programación para utilizar HTML, basta con aprender un par de consas sencillas para empezar acrear paginas

Es MUY FACIL aprender HTML. Tan solo necesitas un editor de texto, un curso de html y un navegador web (como netscape o internet explorer) ni siquiera necesitas estar conectado a internet
